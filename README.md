# Aura: Autonomous University Research Agent

Aura is a multi-agent AI system designed to automate literature discovery, evaluate novelty through cross-domain ideation, and perform rigorous peer-review.

## Core Features
- **The Sentinel:** Daily summaries of new research papers.
- **The Explorer:** Finds structural holes in research and cross-pollinates ideas.
- **The Inquisitor:** Critiques draft papers and verifies citations.
